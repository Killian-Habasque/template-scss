
# DOOTHEME -- MOST USED VARIABLES

## Colors 
```css 
/*
 * primary/ light
 */

var(--wp--preset--color--primary-light)

/*
 * secondary/ light
 */
var(--wp--preset--color--secondary)
var(--wp--preset--color--secondary-light)

/*
 * light/ dark/ white
 */
var(--wp--preset--color--light)
var(--wp--preset--color--dark)
var(--wp--preset--color--white)
```
